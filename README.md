# Pi-hole + Unbound Setup

Detta repo dokumenterar min Pi-hole + Unbound DNS-lösning i proxmox.

## Struktur

- `pihole/` - Pi-hole-relaterade konfigurationer och instruktioner.
- `unbound/` - Unbound-relaterade konfigurationer och instruktioner.

## Installation

1. Följ `pihole/setup.md` för att installera och konfigurera Pi-hole.
2. Följ `unbound/setup.md` för att installera och konfigurera Unbound.
